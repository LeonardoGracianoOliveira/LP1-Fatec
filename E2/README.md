# JavaFX + MySQL CRUD (Alien / Relogio / Portador)

**Descrição**
Projeto exemplo em Java + JavaFX que implementa CRUD (Create, Read, Update, Delete) para três classes:
- Alien (nome:String, planetaDeOrigem:String, dna:int)
- Relogio (modelo:String, cor:String, versao:int)
- Portador (nome:String, aparencia:String, idade:int)

**Instruções**
1. Crie um banco MySQL e execute o script `sql/create_tables.sql`.
2. Ajuste `src/database/db.properties` com suas credenciais MySQL.
3. Compile e rode com Java 11+ e JavaFX disponível no classpath.
4. Exemplo de execução (maven/IDE): rodar a classe `app.Main`.
